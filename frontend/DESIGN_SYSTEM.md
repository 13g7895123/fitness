# UI/UX 設計系統

## 設計理念

這個專案採用**簡約現代**的設計風格，強調：
- ✨ 簡潔的視覺層次
- 🎯 清晰的資訊架構
- 🎨 柔和的色彩搭配
- 📱 響應式設計

## 色彩系統

### 主要色彩
- **Primary**: `#2563eb` - 藍色（主要行動按鈕）
- **Secondary**: `#64748b` - 灰藍色（次要元素）
- **Accent**: `#f59e0b` - 琥珀色（強調）

### 狀態色彩
- **Success**: `#10b981` - 綠色
- **Error**: `#ef4444` - 紅色
- **Warning**: `#f59e0b` - 橙色
- **Info**: `#3b82f6` - 亮藍色

### 中性色彩
- **Background**: `#fafafa` - 淺灰背景
- **Surface**: `#ffffff` - 白色表面
- **Border**: `rgba(0, 0, 0, 0.05)` - 淡邊框

## 組件樣式

### 按鈕
- 圓角：`12px`
- 無文字轉換（保持原始大小寫）
- 字重：`500`（medium）
- 陰影：細膩的懸停效果

### 卡片
- 圓角：`16px`
- 邊框：`1px solid rgba(0, 0, 0, 0.05)`
- 陰影：`0 1px 3px rgba(0, 0, 0, 0.05)`
- 懸停陰影：`0 4px 12px rgba(0, 0, 0, 0.08)`

### 輸入框
- 圓角：`12px`
- 變體：`outlined`
- 密度：`comfortable`
- 聚焦效果：藍色光圈

### 對話框
- 圓角：`24px`
- 最大寬度：依內容而定
- 陰影：深度陰影效果

## 間距系統

- **xs**: `4px`
- **sm**: `8px`
- **md**: `16px`
- **lg**: `24px`
- **xl**: `32px`
- **2xl**: `48px`

## 字體系統

### 標題
- **H3**: `3rem` / `48px` - 主標題
- **H4**: `2.125rem` / `34px` - 次標題
- **H5**: `1.5rem` / `24px` - 區塊標題
- **H6**: `1.25rem` / `20px` - 小標題

### 內文
- **Body 1**: `1rem` / `16px` - 主要內文
- **Body 2**: `0.875rem` / `14px` - 次要內文
- **Caption**: `0.75rem` / `12px` - 說明文字

### 字重
- **Regular**: `400`
- **Medium**: `500`
- **Bold**: `700`

## 響應式斷點

- **xs**: `< 600px` - 手機
- **sm**: `600px - 960px` - 平板（直向）
- **md**: `960px - 1280px` - 平板（橫向）
- **lg**: `1280px - 1920px` - 桌面
- **xl**: `> 1920px` - 大螢幕

## 內容寬度

- 最大寬度：`1280px`
- 居中對齊
- 響應式內邊距

## 動畫原則

- 過渡時間：`0.2s - 0.3s`
- 緩動函數：`ease` 或 `cubic-bezier`
- 保持流暢性和一致性

## 無障礙設計

- ✅ 足夠的顏色對比度（WCAG AA）
- ✅ 鍵盤導航支援
- ✅ 語義化 HTML
- ✅ ARIA 標籤
- ✅ 清晰的焦點指示

## 圖標使用

- 使用 Material Design Icons (MDI)
- 不與文字混用（icon-text 模式已移除）
- 適當的尺寸和間距
- 保持視覺平衡

## 設計原則

1. **簡潔優於複雜** - 減少不必要的裝飾
2. **一致性** - 統一的樣式和行為
3. **清晰的層次** - 明確的視覺優先級
4. **留白的力量** - 適當的間距提升可讀性
5. **回應式優先** - 確保各裝置良好體驗
